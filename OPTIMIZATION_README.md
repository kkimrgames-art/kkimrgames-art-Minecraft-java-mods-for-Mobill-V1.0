# تحسينات أداة تشغيل ماين كرافت على المتصفح

## المشاكل السابقة
كانت أداة التشغيل تقوم بالعمليات التالية في كل مرة:
1. بناء نسخة الويب كاملة (يستغرق وقتاً طويلاً)
2. فحص ومزامنة المودات
3. ضبط نسب ظهور المخلوقات
4. تحميل العديد من الملفات (أصوات، لغات، إلخ)

## التحسينات الجديدة

### 1. خيارات سطر الأوامر
تمت إضافة خيارات جديدة لتجاوز العمليات غير الضرورية:

```bash
# تشغيل اللعبة مع تجاوز البناء إذا كانت النسخة موجودة
python RunWebGame.py --skip-build

# تجاوز فحص المودات ومزامنتها
python RunWebGame.py --skip-mods

# تجاوز ضبط نسب الظهور
python RunWebGame.py --skip-spawns

# دمج عدة خيارات للتشغيل السريع
python RunWebGame.py --skip-build --skip-mods --skip-spawns

# تغيير المنفذ الافتراضي
python RunWebGame.py --port 8080
```

### 2. ملف التشغيل السريع
تم إنشاء ملف `RunFast.bat` الذي:
- يتحقق تلقائياً من وجود نسخة مبنية
- يستخدم خيارات التجاوز إذا كانت النسخة موجودة
- يقوم بالبناء الكامل فقط إذا لزم الأمر

### 3. التحقق الذكي من التغييرات
النظام الأصلي كان يتحقق بالفعل من التغييرات في ملفات معينة، لكن التحسينات تسمح بتجاوز هذا الفحص عندما تريد تشغيل اللعبة بسرعة.

## كيفية الاستفادة

### للتشغيل السريع (بعد أول بناء):
```bash
# استخدم الملف الجديد
RunFast.bat

# أو استخدم سطر الأوامر
python RunWebGame.py --skip-build --skip-mods --skip-spawns
```

### للبناء الكامل (عندما تعدل الكود):
```bash
# استخدم الملف الأصلي
RunWebGame.bat

# أو استخدم سطر الأوامر بدون خيارات التجاوز
python RunWebGame.py
```

## ملاحظات مهمة

1. **المرة الأولى**: يجب أن تقوم ببناء النسخة كاملة في أول مرة
2. **عدم وجود نسخة مبنية**: إذا استخدمت `--skip-build` بدون نسخة مبنية، ستحذر الأداة وستقوم بالبناء
3. **التعديلات على الكود**: إذا قمت بتعديل ملفات Java المصدرية، يجب أن تقوم ببناء النسخة بدون استخدام `--skip-build`
4. **المودات**: إذا قمت بإضافة أو تعديل المودات، لا تستخدم `--skip-mods`

## الفوائد المتوقعة

1. **تسريع التشغيل**: تقليل وقت التشغيل من عدة دقائق إلى ثوانٍ معدودة
2. **توفير استهلاك الإنترنت**: تحميل الملفات مرة واحدة فقط
3. **تقليل استهلاك المعالج**: تجنب إعادة الترجمة في كل مرة
4. **مرونة أكبر**: التحكم في العمليات التي تريد تنفيذها

## استكشاف الأخطاء

### إذا لم تعمل اللعبة:
1. تأكد من وجود ملفات `web-build/classes.js` و `web-build/assets.epk`
2. إذا كانت مفقودة، قم بالتشغيل بدون `--skip-build`
3. تحقق من أن Java مثبت بشكل صحيح

### إذا كانت المودات لا تعمل:
1. لا تستخدم `--skip-mods` عند إضافة مودات جديدة
2. تأكد من أن المودات في المجلد الصحيح (`mods/`)

### إذا كانت نسب الظهور غير صحيحة:
1. لا تستخدم `--skip-spawns` بعد تعديل ملفات الإعدادات
2. شغل `TuneMobSpawns.py` يدوياً إذا لزم الأمر
